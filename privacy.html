Privacy Policy – IC Parser + Query Generator

Last updated: February 2026

Overview

IC Parser + Query Generator is a browser extension that helps users find related Star Citizen Issue Council reports. When a user activates the extension on an Issue Council page, the extension extracts issue details from the page and sends them to a companion API service to return matching results.

Data Collection

The extension does not collect, sell, or share personal data. The extension does not use analytics or tracking tools.

Page Access

The extension accesses content on the active Issue Council tab only to extract issue details (such as title, description, and reproduction steps) to perform a match query. This occurs only when the user interacts with the extension.

Remote Communication

The extension sends extracted issue text to a companion API endpoint operated by the developer to retrieve matching results. Data sent is limited to the issue information visible on the page and is used only to generate the requested results.

The extension does not download or execute remote code. All executable extension code is packaged with the extension (for example, content.js).

Local Storage

The extension may store limited settings locally on the user’s device using Chrome extension storage, such as:

  • user interface preferences (for example, panel visibility)
  • an optional access token/password (if provided by the user) used to authenticate requests to the companion API

This information is stored only on the user’s device and is not sold or shared with third parties.

Data Sharing

The extension does not sell, rent, or share personal data with third parties. Communication occurs only between the user’s browser and the companion API service used to return results.

User Control

Users can remove locally stored settings at any time by clearing the extension’s data or uninstalling the extension.

Changes to This Policy

If the extension’s functionality changes in a way that affects data handling, this privacy policy will be updated.

Contact

For questions about this privacy policy, please contact the developer via the Chrome Web Store listing.

This privacy policy applies solely to the IC Parser + Query Generator browser extension.
