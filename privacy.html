Privacy Policy – IC Helper

Last updated: February 2026

Overview

IC Helper is a browser extension designed to assist users in finding related Star Citizen Issue Council reports by extracting issue details from the currently viewed page and retrieving matching results from a user-configured server.

Data Collection

The extension does not collect, store, or transmit any personal data.

The extension does not:

  • collect personally identifiable information
  • track browsing activity
  • store page content
  • use analytics or tracking services

Page Access

When the user activates the extension, it temporarily reads content from the currently active Issue Council page (such as the issue title, description, and reproduction steps) in order to perform a match query.

This access occurs only after explicit user interaction and is limited to the active tab.

Remote Communication

The extension sends extracted issue text to a user-configured server endpoint solely for the purpose of retrieving matching issue results.

The data sent is limited to the issue content visible on the page and is not retained by the extension.

No executable code is downloaded or executed from remote servers.

Local Storage

The extension may store limited configuration data locally on the user’s device, including:

  • the selected server URL
  • an optional access token for that server
  • user interface preferences

This information is stored using Chrome’s local extension storage and is never shared with third parties.

Data Sharing

The extension does not sell, share, or transfer data to third parties. All communication occurs only between the user’s browser and the user-configured server.

User Control

Users can remove all locally stored data at any time by clearing the extension’s settings or uninstalling the extension.

Changes to This Policy

If the extension’s functionality changes in a way that affects data handling, this privacy policy will be updated accordingly.

Contact

For questions about this privacy policy, please contact the extension developer via the Chrome Web Store listing.

This privacy policy applies solely to the IC Helper browser extension.
